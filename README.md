# Handwritten-Digit-Classifier-MNIST-
Deep Learning Project

## 📌 Introduction
This project implements a deep learning model to classify handwritten digits using the MNIST dataset. The model is built using **TensorFlow/Keras** and includes:
- Multiple hidden layers  
- Batch normalization  
- Dropout for reducing overfitting  

## 📊 Result
The model achieved approximately **97–98% accuracy** on the test dataset.
